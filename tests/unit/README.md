# Dossier de tests unitaires

Ce dossier est destiné aux tests unitaires des fonctions utilitaires, helpers, et logique métier isolée.

- Place ici les tests de fonctions pures, helpers, ou logique métier qui ne dépendent pas d'un contexte serveur ou d'une API externe.

Exemples à tester :
- Fonctions de tri/filtrage de projets
- Helpers de formatage de dates
- Logique de pagination

Ajoute ici les tests unitaires au fur et à mesure de l'évolution du projet.
